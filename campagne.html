Kit de communication livré après publication
Après publication par Into The Shift, vous recevrez un kit de communication finalisé dans votre espace. Ce kit contiendra les éléments prêts à utiliser pour diffuser la campagne.
Vous aurez à disposition : • le lien de passation final, validé et copiable ; • le QR code final généré à partir du lien de passation publié ; • les messages prêts à copier pour le lancement, les relais managers, les relances et les canaux internes ; • le plan de relance conseillé selon vos dates de campagne ; • les affiches, visuels ou supports ajoutés par Into The Shift ; • le lien résultats lorsque celui-ci aura été renseigné par l’équipe Into The Shift.
